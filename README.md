# SpaceUnyship
SpaceUniship est un jeu de tir (shmup) 2D classique et captivant.  Prenez les commandes de votre vaisseau, l'UniShip, et affrontez des vagues infinies d'ennemis pour défendre la galaxie. Accumulez des points, évitez les astéroïdes et testez vos réflexes dans ce jeu simple.  Idéal pour une petite pause rapide !
